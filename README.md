# Automated Stock Trading
Automated stock trading in Python

## Workspace Setup

Clone this repository and run the setup.sh script to add the stock market prediction submodule.

```
sh setup.sh
```

## Repository Structure

This repository uses a [stock market prediction algorithm](https://github.com/intermezzio/stock-market-prediction/) to autonomously buy and sell stocks in Python.
The stock market prediction algorithm is a submodule to allow for independent updating of software.
